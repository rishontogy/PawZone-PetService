# 🗂 3. MEDIA STORAGE FIX (MANDATORY)

---

## ❌ DO NOT store:

* Local file paths (e.g., C:, /tmp/)
* Temporary URLs

---

## ✅ Use one of the following:

---

### Option A (Recommended for Desktop App):

Store as **Base64 or Binary (BLOB)** in database

```js
product.media = [
  {
    type: "image",
    data: base64_string
  },
  {
    type: "video",
    data: base64_string
  }
]
```

---

### Option B:

Store in **app storage folder**

```
/app_storage/media/products/{product_id}/
```

And save path in DB:

```js
url: "/media/products/123/image1.jpg"
```
