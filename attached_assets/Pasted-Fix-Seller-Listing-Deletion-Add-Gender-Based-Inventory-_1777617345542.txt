# 🗑 1. SELLER LISTING DELETE – ERROR FIX

---

## ❌ Problem:

* Seller cannot delete listing
* Shows error (likely authorization or DB issue)

---

## ✅ Required Fix:

---

### 🔹 A. Authorization Check

```js
IF listing.seller_id !== logged_in_user_id:
    RETURN "Unauthorized"
ELSE:

```

### 🔹 B. Safe Delete Logic

---

## Before deleting:

Check:

* If listing is part of active orders

---

### Rule:

* If NO active orders → DELETE directly
* If active orders exist →

```js

```
