# 🧾 3. TRANSACTION CALCULATION FIX

---

## ❌ Problem:

* Platform revenue wrong
* Seller payout wrong
* Transport payout wrong

---

---

## ✅ Fix Calculation:

```js id="txn_fix"
const calculateOrder = (order) => {
  const itemTotal = Number(order.item_total || 0);
  const transport = Number(order.transport_charge || 0);

  const buyerFee = itemTotal > 100 ? 20 : 5;
  const sellerFee = buyerFee;
  const transporterFee = transport > 200 ? 40 : 20;

  const buyerTotal = itemTotal + buyerFee + transport;

  const sellerPayout = itemTotal - sellerFee;
  const transporterPayout = transport - transporterFee;

  const platformRevenue = buyerFee + sellerFee + transporterFee;

  return {
    buyerTotal,
    sellerPayout,
    transporterPayout,
    platformRevenue
  };
};
```
