## 🔹 Spend Calculation:

```js id="buyer_spend"
const totalSpend = orders
  .filter(o => 
    o.buyer_id === currentUserId &&
    o.payment_status === "PAID" &&
    o.status !== "CANCELLED"
  )
  .reduce((sum, o) => sum + Number(o.total_amount || 0), 0);
```
