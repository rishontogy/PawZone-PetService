Build a **full-featured Windows desktop application** named **PawZone**.

This is a **multi-role animal marketplace system** with logistics, payments simulation, and admin control.

The application must be:

* Native Windows compatible (use **Electron + React** OR **.NET (WPF)** preferred)
* Fully functional (not demo UI)
* Clean, modern, responsive UI
* Structured for future backend/API integration

---

# 🧱 TECH STACK (STRICT)

Option 1 (Preferred):

* Electron (desktop shell)
* React + Tailwind CSS (UI)
* Node.js backend (local server inside app)
* SQLite (local database)

OR

Option 2:

* .NET WPF (C#)
* MVVM architecture
* Local database (SQLite)
