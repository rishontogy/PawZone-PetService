### PAWZONE – Route Progress Card for Buyer & Seller Order Details

I want to improve the Buyer and Seller **Order Details** page by adding a visual **Route Progress Card** between the Pickup Point and Drop Point sections.

Do **not** change any backend logic. This is only a UI enhancement using the existing transporter route data.

---

## Placement

Currently the page shows:

Pickup Point

↓

Drop Point

Place the new **Route Progress Card** between these two sections.

It should match the existing PawZone design (green theme, rounded corners, subtle shadow, premium look).

---

## Route Card Design

The card should contain a simplified horizontal S-shaped road similar to the reference image.

Use the PawZone green colour palette.

Inside the road place numbered circular markers.

Do NOT display town names on this small card.

Only show numbered route markers.

Example:

If there are 5 intermediate stops:

① ② ③ ④ ⑤

If there are 10 intermediate stops:

Only display

② ④ ⑥ ⑧ ⑩

This keeps the card compact.

If there are 3 intermediate stops:

① ② ③

If there are no intermediate stops:

Display a message:

"Direct Route"

instead of the road markers.
