# 🔔 2. ALERT TYPES (MUST INCLUDE ALL)

---

## 🕒 Seller Delay

* Trigger: No response in 3 hours
* Night rule applies (after 9 PM → next day 9 AM)

---

## 💳 Payment Delay

* 30 min → Buyer reminder
* 60 min → Admin alert
* 5 hr → Auto cancel

---

## 🚚 Transport Delay

* No transporter assigned in 12 hours

---

## 📦 Delivery Delay

* Delivery not completed within 3 days

---

## 🚫 Cancellation / Failure

* Buyer not paid
* Seller cancelled
* Transporter cancelled

---

## ⚠️ Fraud Alerts

* High cancellation rate
* Repeated reports
* Suspicious activity

---

## 📝 Report Alerts

---

## 🔄 Refund Alerts

# ⚙️ 6. ALERT TRIGGER ENGINE (IMPORTANT)

---

## Implement central function:

```js id="trigger_alert"
function triggerAlert(type, message, order, user, priority="MEDIUM") {
  const alert = {
    id: generateId(),
    type,
    message,
    order_id: order?.id || null,
    user_id: user?.id || null,
    priority,
    status: "ACTIVE",
    created_at: new Date()
  };

  saveAlert(alert);
}
```
